PCOPTRIP - Counting Pairwise Coprime Triples

no tags 

A tuple of three numbers (aa, bb, cc) is called a pairwise coprime triple if gcd(a,b)=1gcd(a,b)=1, gcd(b,c)=1gcd(b,c)=1, and gcd(c,a)=1gcd(c,a)=1.

Let C(n)C(n) be the number of pairwise coprime triples which satisfy 1≤a,b,c≤n1≤a,b,c≤n.

For example, C(3)C(3)= #{(1, 1, 1), (1, 1, 2), (1, 1, 3), (1, 2, 1), (1, 2, 3), (1, 3, 1), (1, 3, 2), (2, 1, 1), (2, 1, 3), (2, 3, 1), (3, 1, 1), (3, 1, 2), (3, 2, 1)} = 13.

Your task is to find C(N)C(N).

Input

First line of Input contains TT (1≤T≤5001≤T≤500), the number of test cases.

Each line of the next TT lines contains a single number NN (1≤N≤1000001≤N≤100000).

It is guaranteed that ∑N≤100000∑N≤100000 in each input file.

Output

For each number NN, output a single line containing C(N)C(N).

Example

Input:

5
1
2
3
10
100

Output:

1
4
13
280
282814

Information

There are 5 input files.

My C++ solution runs in 3.04 sec. (in the worst case)
